What is a Decoder?

A decoder is a combinational logic circuit that converts binary information from n input lines to a maximum of 2ⁿ unique output lines. For each unique input combination, exactly one output line is asserted (activated).

Example: A 3-to-8 decoder has 3 input lines (A, B, C) and 8 output lines (Y0 through Y7). Each of the 8 possible input combinations (000 through 111) activates exactly one output line.

Key Points about Decoders

  • Inputs and Outputs: A decoder with n inputs has 2ⁿ outputs. Common configurations include 2-to-4, 3-to-8, and 4-to-16 decoders.
  • One-Hot Output: Only one output is active (HIGH or LOW depending on the design) at any given time, corresponding to the binary input value.
  • Enable Input: Most practical decoders include one or more enable (EN) inputs. When disabled, all outputs remain inactive regardless of input values.
  • Active-High vs Active-Low:
    • Active-High: The selected output goes HIGH (1), others stay LOW (0)
    • Active-Low: The selected output goes LOW (0), others stay HIGH (1)

Boolean Algebra - Laws

Boolean algebra forms the mathematical foundation for digital logic circuits including decoders.

IdentityA · 1 = AA + 0 = A
Null / DominanceA · 0 = 0A + 1 = 1

CommutativeA · B = B · AA + B = B + A
Associative(A · B) · C = A · (B · C)(A + B) + C = A + (B + C)
