Logic Function Visualizer
ⓘ

How to use:

  • Choose the Decoder Size and Output Type.
  • Type your input variables (they will update on the diagram instantly).
  • Click the minterm buttons to dynamically route wires from the decoder to the logic gate.

Additional Information:

  • Variables supported: W, X, Y, Z, 0, 1 and operators + - ' ( ) ⊕ *.
  • EN (Enable) pin must be either 0 or 1.
  • Active High uses an OR gate to sum the active minterms.
  • Active Low uses a NAND gate and inversion bubbles to achieve the exact same logic function.

Design a circuit by selecting active minterms to generate a Boolean function.
